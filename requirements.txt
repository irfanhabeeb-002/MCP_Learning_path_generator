# Streamlit app + LangGraph agent
# Python 3.11+ required.
#
# Pinning strategy: ~=X.Y.Z (three-component compatible release, per PEP 440).
#
#   ~=X.Y.Z  is equivalent to  >= X.Y.Z, == X.Y.*
#   → allows patch-level updates (security fixes, bug fixes)
#   → blocks minor and major version bumps that regularly break APIs
#
# Why NOT == (exact pin)?
#   Exact pins require a lockfile workflow (pip-tools / poetry) and go stale fast.
#   For a public portfolio project that people git-clone and pip-install once,
#   ~=X.Y.Z is the right balance: reproducible enough, not brittle.
#
# Why NOT ~=X.Y (two-component)?
#   ~=X.Y == >= X.Y, == X.* — this ALLOWS minor version bumps (e.g. 1.3→1.4).
#   In the LangChain ecosystem, minor bumps routinely introduce breaking changes.
#   Two-component pins are under-constrained for this project.
#
# Versions verified working: 2026-06-15
